HTML Interview Questions Day-2

1. Difference between HTML tags and elements

HTML tags are used to hold the HTML element. HTML element holds the content. HTML attributes are used to describe the characteristic of an HTML element in detail. Whatever written within a HTML tag are HTML elements.

2. Explain br tags

The (br) tag inserts a single line break. The (br) tag is useful for writing addresses or poems. The (br) tag is an empty tag which means that it has no end tag.

3. Explain how many different types of lists we have in html

1. Ordered list (OL) In the ordered HTML lists, all the list items are marked with numbers by default. It is known as numbered list also.
2. Unordered list (UL) In HTML Unordered list, all the list items are marked with bullets. It is also known as bulleted list.
3. Description list (DL) The definition list is very appropriate when you want to present glossary, list of terms or other name-value list.
